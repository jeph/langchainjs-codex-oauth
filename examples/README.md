# Examples

Run these from `langchainjs-codex-oauth/` after authenticating.

## Auth

```bash
pnpm auth:login
```

## LangChain hello world

```bash
pnpm example:hello
```

## Tool calling

```bash
pnpm example:tools
```

## LangGraph workflow

```bash
pnpm example:agent
```

## Pick a model

```bash
LANGCHAINJS_CODEX_OAUTH_MODEL=gpt-5.4 pnpm example:hello
```
